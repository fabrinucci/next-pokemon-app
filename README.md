> Pokemon app is a project developed in Next.js that uses the pokeapi to display information about pokemons.

### üè† [Homepage](https://github.com/fabrinucci/next-pokemon-app)

## Install

```sh
yarn
```

## Usage

```sh
yarn dev
```

## Run unit tests
```sh
yarn test:unit
```

## Run end to end tests
```sh
yarn test:e2e
```
